# highline.nvim
Neovim plugin for highlighting the lines

## Installation
### Using packer
```
use 'Blackmorse/highline.nvim'
```

### Using vim-plug

```
Plug 'Blackmorse/highline.nvim'
```

## Usage

Use a command `:Highline` to highlight the line under you cursor!
